# AI Tax Chatbot

An AI-powered Tax Assistant built using Python and Streamlit that helps users calculate income tax, understand tax rules, compare tax regimes, and get smart financial guidance.

---

#  Features

-  Income tax calculation based on slabs
-  Old vs New tax regime comparison
-  Rule-based intelligent chatbot
-  Answers tax-related queries (80C, TDS, ITR, etc.)
-  Tax-saving suggestions
-  Interactive Streamlit UI

---

#  Tech Stack

- Python  
- Streamlit 
- Pandas  
- NumPy 
- Matplotlib  
- Regex (for text parsing)

#  How to Run Locally

## 1 Clone the repository

```bash
git clone https://github.com/your-username/tax-chatbot-new.git
cd tax-chatbot-new
```

---

## 2 Create virtual environment (optional but recommended)

```bash
python -m venv venv
```

Activate it:

### Windows:
```bash
venv\Scripts\activate
```

### Mac/Linux:
```bash
source venv/bin/activate
```

---

## 3 Install dependencies

```bash
pip install -r requirements.txt
```

---

## 4 Run the Streamlit app

```bash
streamlit run app.py
```

---

#  Done!

Your app will open at:

```
http://localhost:8501
```
